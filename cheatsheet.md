This file will contain documentation for all commands available in your game.

Note:  It's a good idea to also make this list available inside the game, in response to a `HELP` command.

help - show a list of all available commands
map -  show the campus map and basic navigation hints
go <location> - moves user to a location on campus, locations could be (dorm, gym, neilson, burton, sabin-reed, hillyer, seelye, botanic-garden, cc)
explore - once the user arrive at certain location, using explore to interact with the place and search for clues
clues - open your clue notebook to check all of the clues the user have found
guess - triggers the dialogue to type the answer of where's the roommate
match - match a new roommate. if form submitted, the user will lose immediately
call parent - call roommate's parent
quit - end the game immediately
